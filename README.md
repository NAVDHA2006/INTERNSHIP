# Data Science Project 1

## Overview

This project performs data cleaning and feature engineering on the given dataset to make it ready for machine learning.

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* OpenPyXL

## Tasks Performed

* Handled missing values
* Detected and treated outliers using IQR
* Created new features
* Performed basic data analysis
* Generated visualizations

## New Features Created

* Year
* Month
* DiscountUsed
* AvgPricePerItem
* CartEfficiency

## How to Run

1. Place the dataset and `project1.py` in the same folder.
2. Open Spyder.
3. Run the script using **F5**.
4. The cleaned dataset will be generated automatically.

## Output

* `cleaned_dataset.csv`

## Conclusion

The dataset was cleaned, analyzed, and transformed into a machine-learning-ready format using Python.
